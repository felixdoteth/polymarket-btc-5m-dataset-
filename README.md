# Polymarket Crypto 5-Minute Markets Dataset

Historical resolution data for all Crypto Up/Down 5-minute prediction markets on Polymarket.

## Dataset

**Files:**
- `polymarket_btc_5m_enriched.csv` — 20,928 markets
- `polymarket_eth_5m_enriched.csv` — 18,969 markets
- `polymarket_sol_5m_enriched.csv` — 18,966 markets
- `polymarket_xrp_5m_enriched.csv` — 18,966 markets

**Total Markets:** 77,829  
**Date Range:** December 18, 2025 — March 16, 2026  
**Source:** Polymarket Gamma API + Binance prices
